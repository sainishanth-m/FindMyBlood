# FindMyBlood

Connecting Patients and Blood Donors in Real Time.

## Project Structure

- `public/`: Static public assets.
- `src/`: Source code.
    - `assets/`: Images, videos, and models.
    - `pages/`: HTML pages (`index.html`, `donate.html`, `need-blood.html`).
    - `css/`: Stylesheets.
    - `js/`: JavaScript files.

## Usage

Open `src/pages/index.html` in your browser to view the application.
